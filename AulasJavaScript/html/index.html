Variáveis

Podem ser declaradas como var ou let e const

Na padronização dos códigos a palavra let substitui o var e o const é usado para dados que não podem ser alerados, ou seja constantes.

Tipos de Variáveis

Utilizando a palavra reservada "Typeof" e possível saber qual o tipo da variável
